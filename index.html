05 — Methodology

How ForestGuard works.

Six steps, from defining the study area to triggering a field alert.

01

Define Study Area

Western Ghats polygon coordinates set as the monitoring boundary.

02

Load Satellite Data

Landsat 8 Collection 2, filtered to ≤10% cloud cover.

03

Preprocessing

Cloud removal, atmospheric correction, reflectance scaling.

04

Compute NDVI

Dense forest reads close to +1.

NDVI = (NIR − RED) / (NIR + RED)
05

Detect Change

NTDDA with adaptive threshold, delta NDVI analysis.

06

Trigger Alert

If Δ NDVI < −0.25 → send SMS + email.
